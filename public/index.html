rescript bindings handbook

This handbook walks you thought the different binding examples regarding Reason and rescript.

In order to get as maximum as possible from these examples, it's recommended to have a vague notion about Reason/ReScript types and TypeScript syntax.

This list focuses on having your code to compile first (even if isn't 100% type-safe), and later explain the path to turn it to type-safe.

JavaScript code we want to generate / ReScript bindings
JavaScript ReScript 
// Call global function
add(1, 2)
 
@bs.val external add: (int, int) => int = "add"
add(1,2)

// Create instance with `new`
const car = new Car()
 
type t
@bs.new external createCar: unit => t = "Car"
let car = createCar()

// Read property
car.color
 
@bs.get external mileage: t => int = "mileage"
car->mileage

// Write property
car.mileage = 1000
 
@bs.set external setMileage: t => int => unit = "color"
car->setMileage(1_000)

// Call function attached to object
car.start()
 
@bs.send external start: t => unit = "start"
car->start

// `this`-based callbacks
car.onStop = function(miles) {
  this.mileage += miles;
}
 
@bs.set external onStop: (t, @bs.this (t, int) => unit) => unit = "onStop"
car->onStop(@bs.this(car, miles) => car->setMileage(car->mileage + miles))
